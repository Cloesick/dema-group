# Pdf Generation

Scripts for generating PDFs, rendering PDF pages, and creating e-reader friendly documentation.

## Scripts in this folder:

- **convert_to_pdf.bat**
- **create_printable_html.py**
- **crosscheck_pdfs.js**
- **generate_ereader_pdfs.py**
- **generate_pdfs_simple.py**
- **render_pdf_pages.js**
- **render_single.js**
- **test_pdf_links.py**

## Usage

All scripts can be run from the project root or from this directory.
