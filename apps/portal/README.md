# DEMA Group Portal

Master portal for the DEMA Group unified industrial platform.

## Features

- Unified product catalog (all companies)
- AI-powered product assistant
- Multi-language support (NL/EN/FR)
- Quote management system
- Customer accounts

## Development

```bash
# From monorepo root
pnpm dev --filter=portal

# Or from this directory
pnpm dev
```

Runs on http://localhost:3001

## Companies Included

- DEMA (pumps, pipes, tools)
- Fluxer (valves, instrumentation)
- Beltz247 (conveyor belts)
- De Visschere Technics (irrigation)
- Accu Components (precision parts)
