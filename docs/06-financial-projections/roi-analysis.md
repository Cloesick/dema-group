## ROI Calculation

### Simple ROI

```
ROI = (Total Benefits - Total Investment) / Total Investment × 100

Total Benefits (5 years):
- Incremental Revenue Margin: €4.56M (Year 1-5 incremental gross profit)
- Cost Synergies: €5.8M (cumulative)
- Total: €10.36M

Total Investment: €1.2M

ROI = (€10.36M - €1.2M) / €1.2M × 100 = 763%

Conservative ROI (using only synergies + margin improvement):
= (€5.8M + €2.1M - €1.2M) / €1.2M × 100 = 558%
```
