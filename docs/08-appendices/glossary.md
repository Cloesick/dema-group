# Glossary of Terms

## Business Terms

| Term | Definition |
|------|------------|
| **B2B** | Business-to-Business; sales between companies rather than to consumers |
| **B2C** | Business-to-Consumer; sales directly to end consumers |
| **COGS** | Cost of Goods Sold; direct costs of producing goods sold |
| **Cross-selling** | Selling additional products to existing customers |
| **EBITDA** | Earnings Before Interest, Taxes, Depreciation, and Amortization |
| **FTE** | Full-Time Equivalent; measure of employee workload |
| **GMV** | Gross Merchandise Value; total value of goods sold through platform |
| **IRR** | Internal Rate of Return; profitability metric for investments |
| **KPI** | Key Performance Indicator; measurable value for success |
| **M&A** | Mergers and Acquisitions |
| **MRO** | Maintenance, Repair, and Operations; supplies for business operations |
| **NPV** | Net Present Value; value of future cash flows in today's terms |
| **NPS** | Net Promoter Score; customer loyalty metric |
| **OEM** | Original Equipment Manufacturer |
| **ROI** | Return on Investment |
| **SKU** | Stock Keeping Unit; unique product identifier |
| **SWOT** | Strengths, Weaknesses, Opportunities, Threats analysis |
| **YoY** | Year-over-Year; comparison to same period previous year |

## Technical Terms

| Term | Definition |
|------|------------|
| **API** | Application Programming Interface; software communication protocol |
| **CDN** | Content Delivery Network; distributed server network for fast content delivery |
| **CI/CD** | Continuous Integration/Continuous Deployment; automated software delivery |
| **CRM** | Customer Relationship Management system |
| **ERP** | Enterprise Resource Planning system |
| **ETL** | Extract, Transform, Load; data integration process |
| **MVP** | Minimum Viable Product; basic version with core features |
| **PWA** | Progressive Web App; web app with native app features |
| **REST** | Representational State Transfer; API architecture style |
| **SaaS** | Software as a Service; cloud-based software delivery |
| **SSO** | Single Sign-On; unified authentication |
| **UAT** | User Acceptance Testing |
| **UI/UX** | User Interface/User Experience |
| **WMS** | Warehouse Management System |
